Task Cards — <task-card>

Background is set on the host element via style="background: var(--card-gradient-N)". Class --lightText or --darkText controls text color.

Kanban Column — <kanban-column>

Columns are self-contained Web Components. Tasks are injected via renderTasks(tasks, renderer). Drag-and-drop is built in but inactive on this page.

Modal Dialog — <modal-dialog>

Three sizes. Close via backdrop click, ESC, or .close(). Never toggle classes directly — always use .open() / .close().

SVG Icons — <svg-icon>

Uses currentColor — inherits parent text color. Set size in px via the size attribute (default 24).

Custom Picker — <custom-picker>

Replaces native <select> elements. Items are injected via setItems([{value, label, color?}]). Fires a change event with {value, label}.

type="color" — Epic color picker
columns="5" / setItems([{value, label, color}])
type="icon" — Category icon picker
columns="5" / setItems([{value, label}])
type="list" size="compact" — Category list
size="compact" / setItems([{value, label}])

Daily Checklist — <daily-checklist>

Reads from and writes to localStorage. Resets daily at the configured reset hour. State on this page shares the "design-system" storage prefix — changes here do not affect your board.

Notes Widget — <notes-widget>

Persists to the server via /api/:profile/notes. On this page the API call returns 404 (no profile alias in the URL) — the widget renders normally but save operations will silently fail.
