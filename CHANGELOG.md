# Changelog

All notable changes to this project are documented here, newest first.

|---------|------------|--------------------------------------------------------------|
| 2.33.1  | 2026-03-06 | Bug fix: Backlog tasks routed to wrong column — `POST /api/:profile/tasks` now accepts optional `status` field in request body and uses it when it matches a valid column ID (falls back to first column otherwise); `createTaskFormSubmitHandler` passes `status: defaultColumnId` to `createTaskApi` so tasks created from the backlog page land in the backlog column. Dev: added `"dev": "node --watch server.js"` script to `package.json` — use `npm run dev` instead of `node server.js` during development so the server auto-restarts on every file save, preventing stale-process bugs. |
